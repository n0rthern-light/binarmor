# BinArmor 
Binary Armor protection Software.

### A proposed build architecture
```
build
├── debug
│   ├── linux
│   │   ├── x86
│   │   └── x86_64
│   ├── osx
│   │   ├── arm64
│   │   ├── fat
│   │   └── x86_64
│   └── windows
│       └── x86_64
└── release
    ├── linux
    │   ├── x86
    │   └── x86_64
    ├── osx
    │   ├── arm64
    │   ├── fat
    │   └── x86_64
    └── windows
        └── x86_64
```
